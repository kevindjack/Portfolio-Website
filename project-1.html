SOC MultiTool

A browser extension (JavaScript) and CLI tool (Batch & PowerShell) that increase efficiency/effectiveness of investigations. When used in unison, 27 tasks are either automated or 60-70% faster.

Project Overview

When used in unison, the following features are available:

  • • IP Abuse Check
  • • IP Information
  • • Tor Relay Search
  • • Hash Reputation
  • • Domain Abuse Check
  • • Domain Information
  • • LOLBin Lookup
  • • Decode Base64
  • • File Info
  • • MAC Address Information
  • • User Agent Parse
  • • Error Code Lookup
  • • Track mean time to resolve a case
  • • Export cases resolved to .txt file
  • • Create 20+ Custom KQL Queries

While I was the sole author of this program, I created it during company time therefore I will not be making the source code publicly available.
